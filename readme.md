```html
<grid-accordion
    :width="960"
    :height="680"
    :columns="5"
    :gutter="4"
    :images="imgs"
    :duration="2000">
```

## 参数
|参数|说明|类型|默认值|可选值|
|----|----|----|----|----|
|width|画布总宽|number|960||
|height|画布总高|number|680||
|columns|列数|number|5||
|gutter|图片间隔|number|4||
|images|图片地址|array|||
|duration|动画播放时间间隔|number|2000||
